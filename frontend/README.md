# Modern Auth UI - Next.js

> [!IMPORTANT]
> **Looking for the full project documentation?** Check out the [PROJECT_GUIDE.md](../PROJECT_GUIDE.md) in the root directory for a complete overview of both Backend and Frontend.

A premium Login and Registration system built with Next.js and Vanilla CSS.

## 🚀 Features

- **Sleek Design**: Modern glassmorphism UI with smooth animations and gradients.
- **Phone Login**: Secure-looking phone number input.
- **OTP Verification**: Multi-digit OTP input with automatic focus shifting.
- **Registration**: Comprehensive form for Name, Number, Address, and City.
- **Responsive**: Works beautifully on all screen sizes.

## 🛠️ Tech Stack

- **Framework**: Next.js 14+ (App Router)
- **Language**: TypeScript
- **Styling**: Vanilla CSS
- **Icons**: Lucide React

## 🏁 Getting Started

### 1. Install Dependencies
```bash
npm install
```

### 2. Run the Development Server
```bash
npm run dev
```
Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## 🔑 Testing the OTP Flow

1. Click on **Login with Phone**.
2. Enter any 10-digit phone number.
3. Click **Get OTP**.
4. **IMPORTANT**: Open your browser console (Right-click > Inspect > Console).
5. The generated OTP will be printed there (e.g., `🔑 YOUR OTP IS: 1234`).
6. Enter the digits in the OTP inputs to log in.

## 📝 Registration
Navigate to the **Register** page from the home screen or login page to see the full registration form.
